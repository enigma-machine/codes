Simple Ciphers

(Start over)

The most well-known type of cipher is the Caesar cipher, named because Julius Caesar used it to send secret messages. This cipher requires a key to decode. It works by shifting each letter forward in the alphabet by the amount determined by the key.

An example of the Caesar cipher

Let's say you wanted to encrypt the text "GREETINGS AND SALUTATIONS" with the key being 3. Each letter would be moved forward three places to obtain the encoded text. G, the seventh letter in the alphabet, would become J, the tenth letter. If you had to encrypt a letter like X, where shifting forward three places would result in a letter after Z, it would loop back around to A.

A Caesar cipher without a key

Of course, to someone besides the intended recipient trying to break a Caesar cipher, they would need to have the message, but they might not have the key. This is where cryptanalysis comes in. This is easy with a Caesar cipher because there are only 25 possible keys, as key 26 iss the same as changing nothing, and any key after is the same as a previous one.
